alert(str) - модальное окно с текстом
prompt(str, [default]) - м. окно с полем для ввода
confirm(str) - м. окно с "да"/"нет"

let - переменная видна внутри функции
var - переменная имеет глобальную область видимости
Если не указано, то по умолчанию - var

"use strict" - объявляется в первой строке документа
Указывает, что надо использовать ECMAScript 5 (ES5), современную версию JS

Строковые преобразования: преобразовываются буквально

Численные преобразования: undefined - NaN, null - 0, true/false - 1/0

Логические преобразования: 0, null, undefined, NaN - false. Иначе - true
("0" = true)

?? - оператор нулевого слияния
